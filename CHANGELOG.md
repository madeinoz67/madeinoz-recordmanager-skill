# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),

## [1.0.0] - 2024-01-19

### Added

- MkDocs documentation system with Material for MkDocs theme
- Complete paperless-ngx integration with full API wrapper
- Country-specific taxonomy system for Australia, US, and UK
- Intelligent document classification and retention rules engine
- CLI tool for document management workflows
- Entity management system (household, corporate, trusts)
- Automated workflow recommendations and compliance monitoring
- Trust document management with ATO compliance features
