# Tutorials

Welcome to the Records Manager Skill tutorials. These hands-on guides will help you get started with managing documents, creating entities, and staying compliant with retention requirements.

## Tutorial Overview

| Tutorial | Description | Time | Difficulty |
|----------|-------------|------|------------|
| [First Upload](./first-upload.md) | Upload your first document to paperless-ngx | 5 min | Beginner |
| [Entity Creation](./entity-creation.md) | Create custom entities for households, trusts, and businesses | 10 min | Intermediate |
| [Retention Check](./retention-check.md) | Verify document retention compliance | 5 min | Beginner |
| [Batch Import](./batch-import.md) | Bulk import multiple documents efficiently | 15 min | Intermediate |

## Before You Begin

Ensure you have:

1. **PAI installed** with the Records Manager Skill loaded
2. **paperless-ngx running** with API access configured
3. **Environment variables set**:
   - `MADEINOZ_RECORDMANAGER_PAPERLESS_URL`
   - `MADEINOZ_RECORDMANAGER_PAPERLESS_API_TOKEN`
