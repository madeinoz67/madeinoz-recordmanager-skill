## How to Use These Examples

Each example includes:
- **Natural Language Prompt**: What you can ask the Records Manager
- **Description**: Expected behavior and outcomes
- **Use Case**: Context and scenario where this is helpful
- **Variations**: Alternative phrasings or related requests

## Tips for Effective Prompting

1. **Be specific**: Include dates, entity names, or document types when possible
2. **Use context**: Reference existing entities or workflows when appropriate
3. **Ask follow-ups**: Use the examples as starting points for more complex requests
4. **Combine categories**: Mix operations (e.g., "Upload and tag these documents")
